# todobook

할 일을 추가하고 관리하는 프로그램입니다.

# 기능
- 할 일 추가
- 할 일 목록 출력
- 할 일 완료 처리
- 완료 항목 수 확인

## 실행 예시
'파이썬 과제 완료하기'이(가) 추가되었습니다.
'Git 실습 지시서 읽기'이(가) 추가되었습니다.
'README.md 작성하기'이(가) 추가되었습니다.

==============================
       할 일 목록
==============================
  1. [ ] 파이썬 과제 완료하기
  2. [ ] Git 실습 지시서 읽기
  3. [ ] README.md 작성하기
==============================
'파이썬 과제 완료하기'을(를) 완료했습니다.

==============================
       할 일 목록
==============================
  1. [✓] 파이썬 과제 완료하기
  2. [ ] Git 실습 지시서 읽기
  3. [ ] README.md 작성하기
==============================

## 사용법
python todo.py
